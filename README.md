# PlatBiBird

Game 2D casual được phát triển bằng Unity và C# theo phong cách Flappy Bird.

## Giới thiệu

PlatBiBird là project mini game được xây dựng nhằm thực hành:
- Physics 2D
- Collision
- UI game
- Quản lý gameplay loop
- Spawn obstacle
- Score system

## Gameplay

Người chơi điều khiển chú chim vượt qua các chướng ngại vật để đạt điểm cao nhất.

## Tính năng hiện có

- Điều khiển nhân vật
- Physics và gravity
- Hệ thống tính điểm
- Spawn obstacle ngẫu nhiên
- Game Over UI
- Restart game
- Animation cơ bản
